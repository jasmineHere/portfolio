Calculator

About project

This project includes all the general mathematical operation like addition, subtraction, division and multiplication that can be performed over two natural numbers. User can also perform operations on integers and decimal numbers.

My role in this project

The development of the following screens of game:
  • a welcome screen where the play button is available, user can click on it & move further in the game.
  • a screen for saving the sessions for every game played and displaying the results of previously played games, so that user can later check, resume the game or can play a new one.
  • the actual tic-tac-toe game screen which appears when user clicks button to play a new one or previously played.
